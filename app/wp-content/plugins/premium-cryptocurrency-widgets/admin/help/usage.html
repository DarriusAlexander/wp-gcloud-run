Basic usage

To add a cryptocurrency widget to a page or post do the following:

  1. Select Premium Cryptocurrency Widgets in the admin menu.
  2. Choose specific widget type (inline, box, table etc), template and color scheme.
  3. Customize widget settings (coin, display currency, real-time feed etc).
  4. Copy the widget shortcode to clipboard.
  5. Paste the shortcode to a page or post.

If you want to add a cryptocurrency widget to one of the WordPress widgets areas you have to install an extra free WordPress plugin (this one or similar), which allows processing shortcodes in widgets. After the plugin is installed get the shortcode as described above and paste it to a widget area.

If you want to call the shortcode from a template you need to wrap the shortcode in a do_shortcode() function call like this: 

<?php echo do_shortcode('[cryptocurrency_widget type="card" template="basic" color="blue" assets="BTC~USD" style="" api="cryptocompare.com" realtime="true" flash="true"]');?>
